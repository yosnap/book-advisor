# UI Agent

## Overview

The UI Agent builds responsive, accessible user interfaces for the book-advisor application. It focuses on creating delightful user experiences across all devices with optimal performance and accessibility.

**Agent ID**: `ui-agent`
**Effort Level**: Medium (claude-sonnet-4-5-20250929)
**Type**: Frontend Development

---

## Responsibilities

- Design responsive component system
- Build reusable component library
- Implement WCAG 2.1 accessibility standards
- Optimize rendering performance
- Manage client-side state and routing
- Implement form validation and feedback
- Create responsive layouts
- Build data visualizations and charts

---

## Technology Stack

**Framework**: Next.js 15 with App Router
**Styling**: Tailwind CSS, CSS Modules
**Components**: Headless UI, Radix UI
**State Management**: React Context, Zustand
**Forms**: React Hook Form
**Validation**: Zod
